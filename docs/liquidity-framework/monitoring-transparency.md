# Monitoring & Transparency

Protocol (TCP) provides **complete transparency** for all liquidity operations, enabling community monitoring and verification.

## What to Monitor

### LP Balance

Monitor the total LP balance:
- Current holdings
- Changes over time
- Withdrawal history
- Verification on-chain

### Lock Status

Monitor the permanent lock:
- Lock expiration date
- Time remaining
- Lock extensions
- Verification on-chain

### Daily Consumption

Monitor daily limit consumption:
- Current day consumption
- Remaining available
- Daily reset
- Verification on-chain

### Pending Proposals

Monitor pending proposals:
- New proposals
- Proposal details
- Timelock status
- Execution time

### Execution History

Monitor execution history:
- Past withdrawals
- Execution dates
- Amounts withdrawn
- Verification on-chain

## Monitoring Methods

### Method 1: PolygonScan

**How to Monitor**
1. Visit PolygonScan
2. Search for Liquidity Manager address
3. View contract events
4. Filter by event type
5. Monitor in real-time

**What You Can See**
- All withdrawal proposals
- All executions
- All cancellations
- Complete history

### Method 2: Contract Functions

**How to Monitor**
1. Call getLPBalance()
2. Call getLockStatus()
3. Call getDailyLimitStatus()
4. Call getTimeUntilExecution()
5. Check proposal status

**What You Can See**
- Current LP balance
- Lock expiration date
- Daily consumption
- Execution time

### Method 3: Community Tools

**How to Monitor**
1. Use LP dashboards
2. Monitor LP metrics
3. Track lock status
4. Analyze patterns
5. Get alerts

**What You Can See**
- Real-time metrics
- Historical data
- Trend analysis
- Alerts and notifications

## Verification Checklist

### Daily Verification

✅ **Check LP balance** — Verify current holdings  
✅ **Check lock status** — Verify lock expiration  
✅ **Check daily consumption** — Verify limit usage  
✅ **Check pending proposals** — Monitor new proposals  

### Weekly Verification

✅ **Review withdrawal history** — Check past withdrawals  
✅ **Assess patterns** — Look for unusual patterns  
✅ **Verify calculations** — Check limit calculations  
✅ **Monitor changes** — Watch for parameter changes  

### Monthly Verification

✅ **Comprehensive review** — Full review of operations  
✅ **Trend analysis** — Analyze trends over time  
✅ **Risk assessment** — Assess operational risks  
✅ **Community feedback** — Gather community feedback  

## Transparency Metrics

### Key Metrics to Track

| Metric | What It Shows |
|--------|---------------|
| **LP Balance** | Total LP holdings |
| **Permanent Portion** | 85% locked LP |
| **Flexible Portion** | 15% available LP |
| **Lock Expiration** | When permanent lock expires |
| **Daily Limit** | Daily withdrawal cap |
| **Daily Consumption** | Current day usage |
| **Pending Proposals** | Proposed withdrawals |
| **Execution History** | Past withdrawals |

### Metric Trends

Monitor trends in:
- LP balance changes
- Withdrawal frequency
- Withdrawal amounts
- Daily limit usage
- Proposal patterns
